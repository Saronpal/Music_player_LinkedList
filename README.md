# 🎵 Sasta Spotify

A lightweight Spotify-inspired music player built using **HTML, CSS, and JavaScript**, with playlist management implemented using a **Doubly Linked List**.

---

## 🚀 Features

- ▶️ Play / Pause songs  
- ⏭ Next & Previous navigation  
- 🧠 Playlist handled using **Doubly Linked List**  
- 🎨 Modern glassmorphism UI  
- 📱 Fully responsive (mobile, tablet, desktop)  
- 🎵 Auto-scrolling song title (like Spotify)  
- 🔁 Scroll animation resets on song change  

---

## 🧱 Tech Stack

- **Frontend:** HTML, CSS  
- **Logic:** JavaScript  
- **Data Structure:** Doubly Linked List  

---

## 🧠 How It Works

- Each song is stored as a **node** in a doubly linked list  
- Each node contains:
  - Song title  
  - Audio source  
  - Cover image  

- Navigation:
  - `next()` → moves to next song  
  - `prev()` → moves to previous song  

- The UI updates dynamically based on the current node
